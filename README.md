# To-do List com Recoil

Aplicação React para gerenciamento de tarefas, utilizando o Recoil para o estado global.

## Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Marcar como concluída
- Remover tarefa
- Filtrar por:
  - Todas
  - Concluídas
  - Pendentes

## Tecnologias

- React
- Vite
- Recoil

1. Instale as dependências:
